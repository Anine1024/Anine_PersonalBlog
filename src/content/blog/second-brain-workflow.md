## 为什么需要第二大脑？

"第二大脑"这个概念由 Tiago Forte 推广开来。核心思想很简单：你的大脑是用来产生想法的，不是用来存储想法的。

我使用 Obsidian 已经两年多，逐步构建起自己的"数字花园"——一个由笔记、想法和项目组成的互联网络。

## 我的文件夹结构

```
📁 Vault
├── 📁 日记/          # 每日笔记和日志
├── 📁 项目/          # 进行中的项目笔记
├── 📁 领域/          # 持续关注的责任区
├── 📁 资源/          # 参考资料
├── 📁 收件箱/        # 快速捕捉
└── 📁 归档/          # 已完成或过时的内容
```

## 快速捕捉的习惯

最重要的实践是快速捕捉：

```markdown
---
tags: [想法, AI-Agent]
created: 2026-05-23
---

# 想法：多 Agent 辩论协议

如果让多个 AI Agent 对一个话题进行辩论
再得出结论呢？类似模拟的同行评审过程。

相关：[[LLM 推理]], [[AI Agent 架构]]
```

## 支撑系统的关键插件

- **Dataview** — 像查询数据库一样查询你的知识库
- **Templater** — 动态模板，保证笔记格式一致
- **Calendar** — 可视化日记导航

> [!tip] 快速上手
> 从每日笔记开始。每天写一句话记录你学到的东西，链接到已有的笔记。就这么简单。

## 总结

你的第二大脑应该和你一起成长。不要执着于"完美的系统"——开始写、开始建立链接，让结构自然浮现。
