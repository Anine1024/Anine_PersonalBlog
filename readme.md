请为我设计一个「个人数字大脑 / AI 技术博客」类型的现代化个人网站 UI 设计稿，要求同时完整覆盖 Web 端与 Mobile 端响应式布局。

# 一、整体定位

网站不是传统博客，而是：

“AI 开发者的个人数字空间（Digital Garden）”

融合以下属性：

- 技术博客
- 数字花园（Digital Garden）
- AI 实验室
- 个人品牌主页
- Obsidian 风格知识库
- 创作者作品集
- 成长记录空间

整体气质：

- 高级感
- 极简
- 科技感
- 沉浸式
- 年轻化
- 有呼吸感
- 强调留白
- 未来感
- 精致卡片式布局
- 不要廉价模板感

参考设计风格：

- Vercel
- Linear
- Raycast
- Notion
- Obsidian
- Apple 官网
- 少量 Stripe 的渐变高级感

页面视觉重点：

“像一个 AI 创作者的第二大脑”

而不是普通 WordPress 博客。
